# evolutionary-antifragile-therapy
Repository for the data and code associated with the manuscript "Evolutionary Antifragile Therapy" by West et. al.

# Manuscript:
Evolutionary antifragile therapy
Jeffrey West, Bina Desai, Maximilian Strobl, Jill Gallaher, Mark Robertson-Tessi, Andriy Marusyk, Alexander R. A. Anderson
bioRxiv 2025.09.11.675645; doi: https://doi.org/10.1101/2025.09.11.675645

# Folder structure:
## Xenographs:
Mouse experiments for the first (even, uneven, high and low dosing protocols) and second (even-uneven, uneven-even switching protocols) experiments

## figures-data
Code (MatLab) associated with any figures in the manuscript which plot data (figures 2, 3, 5, 6, 7).


## figures-theory
Code (MatLab) associated with any figures in the manuscript which show math modeling & theoretical results (figure 1 and supplemental figures).
